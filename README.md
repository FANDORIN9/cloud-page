# Cloud Page — Продуманные веб-решения для вашего бизнеса

**Одностраничный сайт (Landing Page) для digital-студии, которая создает сайты, приложения и бренды, вызывающие доверие и зависть конкурентов.**

## 📌 Общая концепция

Cloud Page — это не просто очередное портфолио веб-студии. Это витрина экспертизы с дерзким, но профессиональным тоном общения. Сайт убеждает клиента в том, что «терпеть старый сайт из 2014 года» больше не нужно, и предлагает решения, которые реально работают на бизнес.

**Ключевые идеи:**
- **Фокус на результате:** «Делаем сайты, за которые не стыдно».
- **Прозрачность:** Честные цены (от 80 000 ₽), открытый процесс работы и реальные цифры (150+ проектов, 98% довольных клиентов).
- **Современный визуал:** Тёмная/светлая тема, анимации, эффект стекла (глассморфизм) и плавный скролл.

## ⚙️ Технологии и особенности

- **Адаптивная вёрстка** — корректно отображается на всех устройствах (от смартфонов до больших экранов).
- **Tailwind CSS + Кастомные стили** — быстрая загрузка и уникальный дизайн.
- **Тёмная / Светлая тема** — автоматическое переключение с сохранением выбранной темы.
- **Intersection Observer API** — плавное появление блоков при скролле.
- **Анимация бегущей строки** (Marquee) для демонстрации услуг.
- **Форма обратной связи** — заготовка для интеграции с Telegram / CRM.
- **Интерактивные карточки** с эффектами наведения и клика.

**Демо-данные:**  
Все цифры (150+ проектов, 98%, 7 лет) и цены (80 000, 200 000, 350 000 ₽) являются демонстрационными и могут быть изменены под реальные кейсы.
